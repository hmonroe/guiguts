Install GuiGuts.

1. Install Perl.
   
   Windows
        1. Download and install ActiveState Perl.
        http://www.activestate.com/activeperl/downloads/

        The page displays the latest version of Perl; x86 and x64.
        Pick one appropriate for computer system. If not sure, download the
        x86 version.

        Find the downloaded file:
        ActivePerl-5.10.1.1007-MSWin32-x86-291969.msi or similar.
        Double-click the msi file and follow the instructions.
        Accept all the defaults.

        The install takes a few minutes.

        2. Get GuiGuts.
        Instructions for setting up Guiguts available here:
        http://www.pgdp.net/wiki/PPTools/Guiguts/Install/Windows

        Links to extra applications are available on this page: image
        viewers, spellcheck etc.

        Some extra Perl install information is also on this page.

    Unixy-types: Linux, Mac OS X, BSD, etc.
        This page has appropriate links in the first section for 
        installation:
        http://www.pgdp.net/wiki/PPTools/Guiguts/Install
